Lightplane documentation

Build instructions

  1. Install lightplane

  2. Install dependencies:

pip install sphinx
pip install myst-parser
pip install sphinx-rtd-theme

  1. Build documentation

cd ${LIGHTPLANE_ROOT}/docs/
sphinx-autogen -o _autosummary index.rst
make html

  1. The documentation can be found under ${LIGHTPLANE_ROOT}/docs/_build/.
